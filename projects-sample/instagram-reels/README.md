# instagram-reels sample project

Copy this folder to wherever you want your IG project to live, then:

1. `cp upload-defaults.yaml.example upload-defaults.yaml` and edit
2. **Make sure your IG account is Creator or Business** (Personal can't post video via web)
3. Drop 9:16 vertical `.mp4` files (≤90 sec) into `videos/`
4. `~/.claude/skills/instagram-upload/scripts/upload-queue.sh <this-dir>`
